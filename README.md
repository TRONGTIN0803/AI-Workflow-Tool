# AI Workflow Tool

A polished SaaS-style frontend for solo developers and creators to design, run, and manage AI-powered workflows.

## Highlights

- Landing page and authenticated application experience
- Dashboard with metrics, execution history, and analytics chart
- Drag-and-drop visual workflow builder with AI Prompt Manager previews
- Workflow templates marketplace with search, filters, save, and duplicate actions
- Execution logs, floating AI assistant, settings, and mock authentication
- Responsive dark-first glassmorphism UI with keyboard command palette (`Ctrl+K`)

## Stack

React, TypeScript, Vite, Tailwind CSS, Framer Motion, Zustand, React Router, Radix UI primitives, Lucide icons, Recharts, and dnd-kit.

## Run Locally

```bash
npm install
npm run dev
```

For a production build:

```bash
npm run build
```
